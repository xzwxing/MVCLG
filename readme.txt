##MVCLG for  "Multi-view clustering based on low-dimensional structure and global representation"

To run our code, please run the Demo_MVCLG.m function.

1. MVCLG.m is our main code.
2. The "data" folder stores the data we used in our experiments.
3. The "function" folder stores the functions we used in our experiments.
4. The "Evaluation" folder stores the clustering evaluation metrics.
